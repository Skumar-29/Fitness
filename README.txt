HEALTH APP – COMPLETE BODY PLAN
Version: health-app-v1.0.0

WHAT IS INCLUDED
- Installable iPhone/Android/Desktop PWA
- Works offline after first successful load
- Six guided 60-minute full-body sessions
- Beginner timing: 35 sec work / 25 sec change-rest
- Intermediate timing: 45 sec work / 15 sec change-rest
- Animated two-position exercise demonstrations
- Exercise library with purpose, target muscles, benefits, steps, easier/harder options and common mistakes
- Vegetarian and vegan-alternative meal plans, including Indian-friendly meals
- Goal modes: general fitness, body-fat reduction, stamina, shape/tone and strength
- Workout history, streaks, optional body check-ins
- Local backup export/import
- Light/dark/system themes
- Safe app-update button and service-worker offline cache

GITHUB PAGES UPLOAD
1. Extract this ZIP.
2. Create a new GitHub repository, or open an existing website repository.
3. Upload index.html, app.js, styles.css, sw.js, manifest.webmanifest and the icons folder to the repository root.
4. Open repository Settings > Pages.
5. Choose Deploy from a branch, select main and /root, then Save.
6. Open the GitHub Pages address in Safari.
7. On iPhone: Share > Add to Home Screen.

IMPORTANT HEALTH NOTE
This app provides general adult health and exercise education. It does not diagnose illness or replace advice from a doctor, physiotherapist, accredited exercise professional or dietitian. Stop exercise for chest pain, faintness, severe breathing difficulty, new neurological symptoms or sharp/worsening pain. Seek urgent medical care when appropriate.

The diet section is a balanced template rather than an individual calorie prescription. People with pregnancy, diabetes, kidney disease, eating disorders, allergies, nutrient deficiencies or other medical needs should obtain individual advice.

WHAT IS INCLUDED
- Six guided workout days with exact 30, 45 and 60-minute session options.
- 55 original short MP4 exercise demonstrations with complete head-to-toe body figures and highlighted working muscles.
- Exercise instructions, benefits, easier/harder options and common mistakes.
- Interactive front/back muscle map.
- Indian vegetarian, vegetarian, vegan and high-protein meal guidance.
- 12-week progression, workout records, measurements and fitness-test records.
- Installable PWA, offline app shell, optional downloadable video packs, backup/import and update check.

IMPORTANT MEDIA NOTE
The included videos are original lightweight anatomy-style procedural animations. They replace the old stick figures and use actual MP4 playback. The video framework is ready for future photorealistic AI-generated clips: replace a matching file in assets/videos and its poster in assets/posters without changing the workout engine.

GITHUB UPLOAD
Upload the CONTENTS of this folder to the repository root so index.html is visible on the main repository page. Do not upload only the ZIP. Enable Pages from main / root.

SAFETY
This is general educational fitness and nutrition content, not medical diagnosis or personalised treatment. Stop exercise for chest pain, faintness, severe breathlessness or sharp/worsening pain and seek appropriate professional help.

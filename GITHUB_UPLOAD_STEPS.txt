FITNESS V6.1 — GITHUB PAGES UPLOAD

This build is for reviewing the High Knees portrait and landscape player before replacing all exercise media.

1. Export a backup from the currently published fitness app.
2. Extract the Fitness V6.1 ZIP.
3. Open the extracted fitness_v6_1_high_knees_review folder.
4. Upload EVERYTHING INSIDE that folder to the root of the GitHub repository.
5. Replace matching files when GitHub asks.
6. Confirm index.html is directly in the repository root.
7. Commit the changes to the Pages branch, normally main.
8. Open the published app while online once so the updated service worker can cache the new files.
9. On the Home screen, tap “Review High Knees screen”.
10. Test both portrait and landscape before approving the design.

Do not upload only the ZIP file.

The app works locally without Firebase. Automatic device-to-device sync still requires the configuration described in FIREBASE_SETUP_GUIDE.txt.
